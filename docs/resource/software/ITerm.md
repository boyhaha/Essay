### iTerm2 快捷键

- | 功能           |                                          |              |
  | -------------- | ---------------------------------------- | ------------ |
  | 新建标签       | cmd + t                                  |              |
  | 关闭标签       | cmd + w                                  |              |
  | 切换标签       | cmd + 数字 / cmd + 左右方向键            |              |
  | 切换全屏       | cmd + enter                              |              |
  |                |                                          |              |
  | 垂直分屏       | cmd + d                                  |              |
  | 水平分屏       | cmd + shift + d                          |              |
  | 切换屏幕       | Cmd + option + 方向键 Cmd + [ 或 Cmd + ] |              |
  |                |                                          |              |
  |                |                                          |              |
  | 选择文本       | cmd + f, tab                             |              |
  | 剪切板历史记录 | Cmd + Shift + H                          |              |
  | 快照返回       | Cmd + Option + B                         | 查看某个时间 |
  | 标签排列切换   | Cmd + Option + E                         |              |
  |                |                                          |              |
  |                |                                          |              |
  |                |                                          |              |
  |                |                                          |              |
  |                |                                          |              |
