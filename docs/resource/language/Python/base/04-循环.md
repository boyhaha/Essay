# 基础 04-循环

* **if 和 while 的区别**
    * if适用于已知循环次数，while适用于未知循环次数；
    * if遍历可迭代对象更简单，while更适合于设置死循环；
    * 其他大部分情况都可互换，没有明显区别。

*  特殊判断条件：
    * 非零即真，零则假

* for\-else
    * 执行过程：
完整的执行完for 循环（每个元素都遍历完，没有break）后，会执行else
    * 使用场景：
遍历可循环序列时，所有元素的条件都不满足，执行else
